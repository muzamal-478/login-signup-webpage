
# Login and Signup Webpage
This is a responsive **login** and **signup** page built using **HTML**, **CSS**, **Bootstrap**, and **JavaScript**.

## Features
- **Login page** with form validation.
- **Signup page** with input fields for user registration.
- **Responsive design** using Bootstrap.
- **Basic JavaScript** for form validation and interactivity.
